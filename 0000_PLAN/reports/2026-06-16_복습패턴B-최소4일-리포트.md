## 1. 1.5배속 반영 — 예, 반영됨
- 강의(영상) 시간에 **×0.7(≈1.5배속, 1/1.5≈0.67)** 적용 후 세션 산정. 본 계산도 동일 기준.
- 단, **랩(hands-on)은 배속 불가** → 실제 시간 그대로(예: TensorFlow Dataset API 2h).

## 2. 패턴 B 정의(적용)
- **실습 세션 1개 → 복습 세션 1개**(1:1).
- **강의 세션 2개 → 복습 세션 1개**(2:1).
- 복습 세션 = (실습 세션 수) + ⌈강의 세션 수 ÷ 2⌉.

## 5. 트레이드오프 정리
- **장점(B)**: 복습이 일자 단위로 확실히 확보 → 정착·기억 강화. "1실습:1복습" 리듬 명확.
- **단점(B)**: 총 ~25~28주(12월), 이전 목표(18주/10월 말)와 크게 어긋남. 랩 1:1 복습이 일수를 크게 늘림.
- 절충(C, 2콘텐츠:1복습)이면 ~22주(11월 말)로 중간.
