- 날짜: 2026-06-15
- 작업자: Claude (AI)
- 대상: `PMLE_18week_study_plan.html`
- 요청: 일간 활동을 "글자"로만 두지 말고 **실제 실습 링크 + 작업 방법**을 제공하고, 용어를 클릭하면 **해당 작업 공간으로 바로 진입**하도록 처리(HTML 전체 적용). RULE.md에도 규칙 추가.

---

## 1. 구현 내용

1. **용어 자동 하이퍼링크 (`linkify`)**
   - `LINKS` 매핑(40+ 항목: 제품/콘솔/문서)을 기준으로, 본문 `.day-act`·`.wk-theme`와 드로어(학습 카드·자료)에서 제품/콘솔/랩 용어를 자동으로 `<a target="_blank">`로 변환.
   - 단일 패스(최장 일치 우선) 처리로 링크 중첩/오염 방지.
2. **드로어 강화**
   - 상단 **"🚀 작업 공간 · 바로가기"** 칩(그날 카드에 등장한 콘솔/문서 바로가기).
   - **작업 방법 안내 노트**: 학습 카드가 작업 순서·산출물임을 안내, 링크 색상 의미 설명.
   - 자료(`resources`): `Lab:` → 🧪 Skills Boost 검색, `Doc:`/기타 → 제품 콘솔 링크 + 🔎 공식 문서 검색.

## 2. 링크 대상 규칙 (할루시네이션 방지)

| 종류 | URL 형식 | 비고 |
|---|---|---|
| 콘솔 작업 공간 | `https://console.cloud.google.com/...` | 안정적 상위 경로 사용(bigquery, vertex-ai/workbench, model-garden, feature-store, pipelines, models, online-prediction 등) |
| 실습 랩 | `https://www.cloudskillsboost.google/catalog?keywords=<랩명>` | 검증된 검색 파라미터(고정 lab ID 미사용 → 오류/폐기 위험 회피) |
| 공식 문서 | `https://www.google.com/search?q=<용어> site:cloud.google.com` | 정확 doc 경로 불확실 시 공식 사이트 검색으로 안전 연결 |

- 정확한 deep-link 경로가 불확실한 경우(예: Model Armor, Vision/Translation API, Ray, Vizier, VPC Service Controls)는 **추측 경로를 만들지 않고** 공식 문서 검색으로 연결.
- 구 `vertex-ai` 콘솔 경로는 **Agent Platform으로 자동 리다이렉트**될 수 있음을 드로어 노트에 명시.

## 3. UI 영향 (RULE §5)

- 추가만 수행(색/레이아웃/타이포 미변경). 링크는 밑줄+색으로 표시: 콘솔(파랑), 랩(초록), 문서(보라).
- 신규 CSS 클래스: `.dl/.dl-console/.dl-lab/.dl-doc/.ql/.ql-wrap/.res-doc`.

## 4. 검증

- `node --check` JS 문법 통과. `LINKS` 40개 매핑·드로어 연결(quickLinks/resLink)·본문 자동링크 적용 확인.

## 5. 후속 권장

- 콘솔 deep-link 경로는 Agent Platform 정식 경로로 추후 갱신 가능(현재는 리다이렉트 의존). 새 제품/용어 추가 시 HTML 내 `LINKS` 배열에 등록.
