## 1. 한눈에 보는 골격
- **총 22개 주차 탭** = 과거(완료·복습용) **4** + 현재 학습 **15** + 시험대비 **3**. + 개요 탭 + 📦 보충 탭.
- 좌측 레일이 단계별로 그룹화: `✅ 이전` / `▶ 현재 학습`(코스별 구분선) / `🎯 시험 대비` / `📦 보충`.
- 모든 현재 주차 = **주 4일 × 일 1.5h** 기준(긴 랩 날은 ⤴ 상향).

## 4. 주차 1개의 내부 구성(현재 학습 주차)
1. **헤더**: 파트 칩 + 코스명(영문) + 한글 요약 + 상태 배지(✅/⏳/⬜) + 코스 링크.
2. **🎬 강의 상세**(접이식): 코스의 들어야 할 **모듈/영상/자료 목록 + 링크**, ★=핵심 모듈(개요·요약은 무표시). 출처 LINK.html.
3. **일자 카드 4개**: 실습(🧪 랩 링크+진행배지) / 강의(★모듈 우선) — 줄별 `[Nm]` 시간칩 + `⏱ 합계` + (2h 랩) ⤴ 상향 + `📋 학습 자료` 버튼.
4. **실습별 복습 카드(수시·무날짜)**: 실습마다 1개. 시간 배분 + `📌 핵심 포인트 6`(접힘) + `❓ 퀴즈 5문항`(접힘·보기 선택형·정답/오답·선택지별 이유).
5. **5일차(선택)**: 여유 시 밀린 랩·Challenge·심화.
6. **복습일(결석 시)**: 팀원 결석 시 진도 대신 능동 회상·랩 재실행·퀴즈·teach-back.

## 5. 콘텐츠 자산(현재)
- **실습 링크 127개**(/labs/) — LINK.html 정확 href(RULE §9), 로그인 시 열림.
- **복습 카드 25개**(실습 18 + 강의전용 주차 일반 7), **퀴즈 90문항**(18×5, 보기 선택형·선택지별 이유), **핵심 포인트 18×6**.
- **강의 상세 14블록** — 코스별 모듈/영상 수백 개 링크.

## 6. 진행률·이식성
- 진행률 = **DOM 기반**, 숫자 일자(data-day=1~4)만 집계. 수시 복습(qN)·5일차(f)·복습일(r)은 **미포함**(부족 시 체크용).
- 과거 4주는 INITIAL_PROGRESS로 100% 완료 표기.
- 단일 HTML 이식성: 저장/내보내기/가져오기 + `INITIAL_PROGRESS` 임베드(localStorage 없이도 동작).
