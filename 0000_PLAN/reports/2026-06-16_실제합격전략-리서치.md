# 리포트 — 실제 PMLE 합격 전략 + 우리 플랜 적용 (딜레마 해소)

- 날짜: 2026-06-16
- 작업자: Claude (AI)
- 질문: "코스를 안 할 수도 없는데 비중은 안 맞고 애매하다. 남들은 어떻게 성공 전략을 짜나?"
- 답(요지): **코스는 다 완주하되, "모의고사가 학습을 끌고 가게(practice-test-driven)" + 약점·고비중 집중**. 중요도는 "뺄 것"이 아니라 "더 깊이 팔 곳"을 정함.

## 1. 조사된 실제 합격 전략(공통점)
1. **Skills Boost 경로(코스) 완주** — 빼지 않음.
2. **모의고사 주도 학습**: 문제은행 **2~3회 전체 풀이**, **도메인별 정답률 추적 → 약점에 시간 추가.** (완주 ≠ 목표, 약점 제거 = 목표)
3. **강의 1.25~1.5배속 선시청 → 랩**(우리 ×0.7 반영과 일치).
4. **타임드 풀모의 + 케이스/서비스 선택 문제**로 실전화.
5. **생성형 AI(Model Garden·파운데이션 모델 선택/파인튜닝·RAG) 집중**(개편 후 크게 출제).
6. 현실 기간 **8~10주**(경험자 30일도), 합격선 ~70%, 50~60문항/2시간/$200.
7. 덤프 X, 구조화 문제은행 O.

## 2. 우리 플랜에 적용 (딜레마 해소안)
"코스는 다 한다 + 중요도는 '깊이'로 반영 + 모의가 주도" 를 결합:

- **A. 기초 완주(현재 W1~)**: 경로 코스 완주(강의 1.5배속 + 랩). — 지금 플랜 유지.
- **B. 조기 진단 모의**: 기초 한 바퀴(또는 핵심 도메인) 후 **풀모의 1회**로 약점 도메인 식별.
- **C. 약점·고비중 집중 루프**: 매주 미니 퀴즈 + 2~3주마다 모의 → **약점(특히 고비중 S3·S4·S5 + 생성형)에 복습/추가 랩 투입.** (Keras 등 저우선·강한 영역은 더 안 판다.)
- **D. 시험대비 단계(2~3주)**: 풀모의 2~3회 + 케이스/서비스 의사결정표 + 약점 마감. **고비중·생성형 중심.**

→ 즉 **재배분(코스 시간 조정)보다 "연습 레이어 + 약점 집중"을 얹는 것**이 정석. 코스는 다 하되, 시간이 남거나 약점이면 고비중에 더 쓰는 구조.

## 3. 일정 영향
- 기초 완주 ~17~18주(콘텐츠 78h + 복습) + **시험대비 2~3주** = **~20주** → 종료 ~11월 초~중순, 응시 11월 중·하순.
- 모의/퀴즈는 기존 주차의 리뷰 슬롯·복습일에 흡수 → 큰 추가 부담 적음.

## 4. 구현 제안(승인 시)
1. 각 주 리뷰일/복습일에 **도메인 미니 퀴즈** 명시(이미 퀴즈 라인 있음 → 도메인 태깅).
2. 기초 후 **진단 모의 주차** 1개 삽입.
3. 끝에 **시험대비 단계(모의×2~3 + 케이스 + 의사결정표 + 약점)** 추가.
4. 약점·고비중(S3·S4·S5·생성형) 보강 칸을 5회차/복습일에 우선 배치.
5. 기존: 복습 B(시간 정합)·모든 주 ≥4세션·랩 링크(§9)·1.5배속 유지.

## 5. 한계/주의
- 모의 "문제은행"은 **공식 샘플 + 합법적 구조화 문제집** 사용(덤프 금지) — 본 플랜엔 링크/자리만 마련, 실제 문제 콘텐츠는 사용자가 선택.
- "약점 집중"은 모의 결과에 의존 → 진단 모의 전엔 비중 추정으로 시작.

## 6. 결정 요청
- 이 "완주 + 모의주도 + 약점/고비중 집중 + 시험대비 2~3주" 구조로 **플랜에 반영(구현)**할지?
- 그러면 앞서 정한 복습 B(시간 정합)·모든 주 ≥4세션과 함께 한 번에 재빌드하겠습니다(종료 ~11월 중순).
