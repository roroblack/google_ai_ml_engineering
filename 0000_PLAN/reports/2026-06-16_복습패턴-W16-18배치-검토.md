# 리포트 — W16~18 배치 이유 + 복습 패턴 결정 + 시험대비 누락

- 날짜: 2026-06-16
- 작업자: Claude (AI)
- 성격: **리포트만(HTML 미변경).** 카드 범위는 "현재 주차만"으로 확정(과거 완료 주차엔 5회차·복습일 미부착) — 구현 시 반영.

## 1. W16/17/18 배치 이유 (질문 답변)
- **실제 path 17 커리큘럼의 마지막 3개 코스 = Responsible AI 3종**이라 그 순서를 그대로 따름:
  - W16 Responsible AI: Fairness & Bias (4h)
  - W17 Responsible AI: Interpretability & Transparency (3h)
  - W18 Responsible AI: Privacy & Safety (5h)
- 각 3~5h로 짧아 1주씩 배정 → 자연히 마지막 3주.

## 2. 발견된 문제 — 시험대비/모의고사 단계 누락 (중요)
- 옵션 B(실제 코스 순서 재정렬) 때, **원래 플랜의 "시험 모드(모의고사 5회·약점보강·서비스 의사결정표)" 구간이 사라짐.**
- 현재는 마무리가 **Responsible AI 학습으로 종료** → **모의고사·실전 대비가 플랜에 없음.** 합격 준비로는 공백.
- 또한 path 17엔 **"Build a Certification Study Guide: PMLE"(1171)** 코스가 있는데 현재 플랜에서 빠져 있음.

### 권장
- W18 **이후에 "시험 대비" 단계(2~3주) 추가**: 1171 Study Guide + 공식 샘플 + 모의고사 + 서비스 의사결정표 + 약점 도메인 보강.
- 이러면 종료가 **11월 중순**으로 이동(학습 18주 → 10월 말 + 시험대비 2~3주).

## 3. 복습 패턴 — 미결(결정 필요)
지금 빌드엔 "1실습:1복습 / 2강의:1복습" **일자 교차 패턴이 미적용**(랩 연속 + 마지막 날 복습 라인 + 주간 복습일 카드만). 적용 시 18주와 충돌:

| 옵션 | 복습 처리 | 현재 주차 | 종료 |
|---|---|---|---|
| **A**(권장) | 세션 내 복습(매 실습일에 정리 시간 포함) + 주1 복습일 카드 | 18주 | ~10월 말 |
| B | 별도 복습일(엄격 1:1·2:1) | ~30주 | ~2027-01 |
| C | 절충(2콘텐츠:1복습일) | ~22~24주 | 11월말~12월 |

- A: 복습이 "별도 날"은 아니지만 **모든 날에 복습 시간 명시**(예: 랩 60m + 🔁복습 30m = 90m). 18주 유지.
- B: 패턴 그대로지만 분량(42랩+90h)상 ~30주로 크게 늘어남.
- → **A + 시험대비 2~3주**가 현실적(10월 말 학습 종료, 11월 중순 응시) 추천.
