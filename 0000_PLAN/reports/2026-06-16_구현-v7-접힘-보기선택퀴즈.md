## 1. 핵심 포인트·퀴즈 기본 접힘
- `📌 핵심 포인트 N개`, `❓ 퀴즈 5문항` 각각 `<details>`로 **기본 접힘**. 복습 카드가 평소 깔끔하고, 헤더 클릭 시 펼침.

## 2. 퀴즈 3단계 순환 클릭
- 질문 카드 클릭: **0(질문만) → 1(보기) → 2(정답·해설) → 0(접힘)** 순환.
- 하단 힌트가 단계별 안내: "클릭하면 보기" → "보기 선택 또는 클릭하면 정답·해설" → "정답·해설 표시됨(클릭하면 접기)".
- 카드별 **⊕ 모두 펼치기 / ⊖ 모두 접기**.

## 3. 보기 선택형 + 정답/오답 + 선택지별 이유
- 보기(1~4)를 **클릭 선택** 가능. 선택 시 정답 공개 단계로 전환.
  - 정답 보기 → 초록 **✓ 정답**, 내가 고른 오답 → 빨강 **✗ 내 선택(오답)**.
- **선택지마다 맞고 틀린 이유(why)** 를 정답과 함께 노출: 정답은 왜 맞는지, 각 오답은 왜 틀린지 1문장.
- 이를 위해 퀴즈 데이터 재생성(실습당 5문항 × 선택지 4개 each `why`).

## 4. 기술
- 콘텐츠: 18개 실습 × {포인트 6, 퀴즈 5(선택지4·정답·why4), 시간배분} — 서브에이전트 3개, 안정 개념 기반(환각 방지), `node`/스키마 검증.
- UI: `<details>`(native) + `.rq[data-stage]` CSS 단계 표시. JS = 이벤트 위임 1개(qx 버튼 · 보기 li 선택 · 질문 순환). `</body>` 앞 독립 script.
- 유지: 강의 상세 14블록·랩링크 127(§9)·진행률(복습 제외)·중요도가중·보충패널·시간칩.
